# FoodExpress

A full-stack food delivery platform inspired by Uber Eats.

## Tech Stack

- Java
- Spring Boot
- MySQL
- React
- Docker

## Core Features

- Restaurant browsing
- Menu management
- Shopping cart
- Order management
- Merchant dashboard
